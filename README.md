# AI Study Assistant

A simple AI-powered study chatbot built with Streamlit and Groq (LLaMA 3.3).

---

## Live Demo
https://aistudyassistant-kssz6n7haurnqkhlawsgpg.streamlit.app/

## Features

- Chat with AI about any topic
- Powered by LLaMA 3.3 70B via Groq
- Fast responses
- Chat history maintained during session

---

## Tech Stack

- Python
- Streamlit
- Groq API
- LLaMA 3.3 70B
